## 2. VISION & MISSION

### Vision long-terme
Clemencia Garden est un **démonstrateur local** de production maraîchère biologique, documentée et pilotée par les données, dans un contexte tropical insulaire (Maurice). Le projet prouve qu'une **réduction partielle des dépendances alimentaires** est possible via un design écologique rigoureux et une gestion frugale (sans électricité, eau limitée, capital réduit).

### Mission TikTok
Montrer le **réel agricole** à petite échelle : expériences, données, réussites ET limites. Pas de promesses théoriques, uniquement la preuve terrain. Construire une **communauté autour de la souveraineté alimentaire locale** et de l'agroécologie tropical.

---

## 3. CONTEXTE GÉOPOLITIQUE & ENJEUX

### Pourquoi Maurice ?
- **80% des aliments importés** → dépendance alimentaire chronique
- **Climat tropical instable** → stress hydrique croissant, cyclones réguliers
- **Sols fragiles** → lessivage intense, érosion post-pluies
- **Ravageurs tropicaux** → pression sanitaire constante
- **Attente sociétale émergente** → mieux comprendre d'où vient la nourriture, valoriser les savoir-faire locaux

### Le défi Clemencia
Produire **localement, bio, résilient, économiquement viable** sur **1 arpent (~4 000 m²)** sans accès immédiat à eau réseau ni électricité. Convertir une **contrainte en avantage** via design frugal et tech données.

## 4. TROIS COMPOSANTES DU PROJET

### A. Production maraîchère bio
- **Surface** : ~1 arpent (4 000 m²), dont 500 m² planches effectives au départ
- **Modèle** : Maraîchage bio-intensif + permaculture pratique + SPIN (Small Plot Intensive Nutrition)
- **Cultures prioritaires** :
  - Aromatiques : basilic, coriandre, persil, menthe, citronnelle
  - Brèdes & greens : pak choi, laitue, brèdes locales
  - Légumes robustes : piment, aubergine, gombo, tomate, concombre
  - Petits fruitiers : papayer, bananier, goyavier, avocatier
  - Médicinales : Tulsi, Neem, Aloe Vera, Moringa
- **Rotation & cycles** : 30-90 jours par culture ; minimum 3 cycles/an
- **Rendement cible** : 6-12 tonnes/arpent/an (bio-intensif tropical)

### B. Bibliothèque de semences
- **Objectif** : Documentation progressive de variétés locales & résilientes
- **Format** : Fiches techniques, photos, données de culture, généalogie
- **Cadre légal** : Respecter Plant Protection Act 2006 (Maurice), NPPO requirements
- **Valeur ajoutée** : Vente de semences (revenue secondaire) + conservation du patrimoine génétique

### C. Média de preuve terrain
- **Contenu** : Journal visuel du projet (photos, vidéos courtes, dataviz, récits)
- **Plateformes** : TikTok (@clemenciagarden), Instagram, YouTube, Patreon
- **Ton** : Authentique, sobre, montrant réussites ET limites. Pas de marketing creux.
- **Fréquence** : 3-5 posts/semaine minimum (TikTok)
- **Audience cible** : Fermiers locaux, hobbyistes jardin, étudiants agro, ONG résilience, entreprises agri-tech

## 5. CONTEXTE CLIMATIQUE & CONTRAINTES OPÉRATIONNELLES

### Climat tropical humide (Maurice, région Clemencia)
| Paramètre | Valeur |
|-----------|--------|
| Température | 25-30°C année |
| Saison pluies | Décembre-avril (300+ mm/mois à pics) |
| Saison sèche | Mai-novembre (50-100 mm/mois) |
| Humidité relative | 74-82% (constant, risque fongique) |
| Cyclones | Décembre-mars, classe III (>120 km/h) |
| Évapotranspiration | 2 mm/jour (ombrage), 4 mm/jour (plein soleil) |

### Contraintes de départ (Clemencia)
- **Eau** : Pas d'accès réseau. Stockage pluie ~86 m³ requis pour mois sec.
- **Électricité** : Aucune. Irrigation gravitaire + capillaire obligatoire.
- **Capital** : ~€1,255 démarrage (Rs 68,000) ; ~€443/mois charges (Rs 24,000)
- **Logistique** : Scooter seulement. Pas de transport lourd immédiat.
- **Main-d'œuvre** : Kush + occasionnel. Optimiser l'efficacité temps/travail critique.

## 6. APPROCHE AGROÉCOLOGIQUE & TECH

### Principes fondamentaux
1. **Santé du sol = fondation** : Compost, matière organique, paillage systématique, agroforesterie
2. **Prévention avant traitement** : Design cultural (rotations, biodiversité, couverture)
3. **Bio only** : Neem, soufre, huiles essentielles, extraits biologiques (nada synthétique)
4. **Eau précieuse** : Collecte pluie, citernes, irrigation au goutte-à-goutte basse pression, capillaire
5. **Tech frugale** : Données sans électricité (Bash, Git, Google Sheets, IA assistée)

### Stack technologique
- **Tracking** : Google Sheets (culture, dates, irrigation, ravageurs, rendement, coûts)
- **Versioning** : Git + GitHub (historique décisions, protocoles reproduisibles)
- **Analyse** : Bash scripts + Claude/Gemini pour insights données
- **Décision** : Agents IA (Gemini, Claude) pour diagnostic ravageurs, ajustements irrigation, optimisation rendement

## 13. TERMINOLOGIE & GLOSSAIRE TIKTOK

### À utiliser systématiquement
- **Bio-intensif** : Densité élevée, rendement max sur petit espace
- **Permaculture** : Design écologique permanent, multi-étages
- **Agroécologie** : Agriculture + écologie, prévention, biodiversité
- **SPIN** : Small Plot Intensive Nutrition farming
- **Compost** : Valorisation déchets organiques
- **Paillage** : Couverture sol (réduit eau, érosion, adventices)
- **Ravageurs** : Insectes/maladies problématiques
- **Rendement** : Masse/volume récolte par surface
- **Tropical** : Climat chaud-humide, spécificités Maurice

### À éviter (clichés TikTok)
- "Révolution agricole"
- "Sauver la planète" (trop vague)
- "Facile et rapide" (faux, décourage)
- Jargon académique sans explication
- Hype infontrée
